Formats the cstr using the libfmt library.

See here for details: https://fmt.dev/latest/syntax.html

Note that compile-time format check is performed, and hence cstr needs to be a string literal. If no compile-time check is required, and/or the string is a variable, call FARM_RUNTIME_FORMAT(fmt, ...) instead.

FARM_COMPILE_TIME_FMT is not defined, the runtime version will be used, to speed up compile times.
