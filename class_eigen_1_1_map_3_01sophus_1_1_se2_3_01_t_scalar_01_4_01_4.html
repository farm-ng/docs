Detailed Description

template<class TScalar>
class Eigen::Map< sophus::Se2< TScalar > >

Specialization of Eigen::Map for Se2; derived from Se2Base.

Allows us to wrap Se2 objects around POD array.

Group action on 2-points.

This function rotates and translates a two dimensional point p by the SE(2) element bar_transform_foo = (bar_R_foo, t_bar) (= rigid body transformation):

p_bar = bar_R_foo * p_foo + t_bar.
